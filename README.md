# SVapp

1. Классическая формула (Оригинальная, 1962 год)В своей книге «Анализ ценных бумаг» Грэм предложил формулу для расчета внутренней (справедливой) стоимости акций роста (V):

V = EPS*(8.5 + 2g)

EPS (Earnings Per Share): Прибыль на одну акцию за последние 12 месяцев (TTM).
8.5: Базовый коэффициент P/E для компании с нулевым ростом.
g: Ожидаемый среднегодовой темп роста прибыли на ближайшие 7–10 лет (в процентах, т.е. если рост 5%, подставляем просто 5)

2. Модернизированная формула (1974 год)
Со временем Грэм понял, что формула не учитывает альтернативную стоимость денег — процентные ставки на рынке облигаций. Если государственные облигации приносят высокий доход, акции должны стоить дешевле, и наоборот.

Он обновил формулу, добавив в неё текущую доходность высококлассных корпоративных облигаций (AAA):

V =((EPS*(8.5 + 2*g))*4.4)/Y

4.4: Средняя доходность корпоративных облигаций с рейтингом AAA в 1962 году (когда была создана первая формула). Это была его «безопасная» базовая ставка.
Y (Yield): Текущая доходность первоклассных корпоративных облигаций AAA в процентах.

Замечание для проекта: Сегодня вместо редких облигаций AAA часто подставляют текущую доходность 10-летних государственных облигаций США (10-Year Treasury Rate)

3. Число Грэма (Graham Number)
Это еще один его знаменитый критерий, но уже для защитных (консервативных) инвесторов из книги «Разумный инвестор». Грэм считал, что цена акций (P/E) не должна превышать 15, а отношение цены к балансовой стоимости (P/B) не должно быть больше 1.5.Перемножив эти коэффициенты (15*1.5 = 22.5), он вывел формулу максимальной справедливой стоимости:

V = sqrt(22.5*EPS*BVPS)

BVPS (Book Value Per Share): Балансовая стоимость компании на одну акцию.Если текущая рыночная цена акции ниже этого Числа Грэма — компания считается недооцененной

Pабрать для выбранного тикера всего 4 параметра:
Current Price (Текущая цена акции — чтобы сравнить её с результатом расчета).
EPS (Прибыль на акцию).
BVPS (Балансовая стоимость на акцию — нужна для Числа Грэма).
Expected Growth (g) (Ожидаемый рост — аналитики Yahoo обычно публикуют прогноз роста на ближайшие 5 лет во вкладке Analysis).
Плюс, для второй формулы нам понадобится одна константа — текущая ставка доходности облигаций (её можно зашить жестко в код или тоже парсить, например, тикер ^TNX — доходность 10-летних облигаций США).
